# Estad√≠stica descriptiva


En este v√≠deo, se aborda el tema de la **estad√≠stica descriptiva**, una parte fundamental del an√°lisis exploratorio de datos. Antes de dedicar tiempo a crear modelos complejos, es crucial explorar los datos de manera sencilla y eficiente. Una forma de hacerlo es calcular estad√≠sticas descriptivas para obtener un resumen general de las caracter√≠sticas b√°sicas del conjunto de datos. El an√°lisis estad√≠stico descriptivo ayuda a resumir los datos mediante medidas clave, como la media, la desviaci√≥n est√°ndar, los cuartiles y los valores at√≠picos, proporcionando as√≠ una visi√≥n m√°s clara de la distribuci√≥n de las diferentes variables.

Uno de los m√©todos m√°s √∫tiles en Python para obtener un resumen de los datos es la funci√≥n `describe()` de pandas. Al aplicarla a un dataframe, esta funci√≥n calcula autom√°ticamente las estad√≠sticas b√°sicas para todas las variables num√©ricas. Entre las estad√≠sticas que muestra est√°n la media, el n√∫mero total de puntos de datos, la desviaci√≥n est√°ndar, los cuartiles y los valores extremos. Adem√°s, los valores faltantes (NaN) se omiten autom√°ticamente. Esto proporciona una visi√≥n general de c√≥mo se distribuyen los datos y qu√© caracter√≠sticas son m√°s relevantes. Tambi√©n podemos encontrarnos con variables categ√≥ricas, como la variable de tracci√≥n en un conjunto de datos de coches. Estas variables tienen valores discretos y se pueden dividir en diferentes categor√≠as. Un ejemplo en el conjunto de datos de coches es el sistema de tracci√≥n, que puede ser tracci√≥n delantera, trasera o a las cuatro ruedas.

Para resumir los datos de las variables categ√≥ricas, podemos usar la funci√≥n `value_counts()`. Esto nos da la frecuencia de cada categor√≠a. Por ejemplo, si aplicamos `value_counts()` a la variable "tracci√≥n", podemos ver cu√°ntos coches hay en cada categor√≠a: 118 coches con tracci√≥n delantera, 75 con tracci√≥n trasera y 8 con tracci√≥n a las cuatro ruedas. Esta funci√≥n facilita la comprensi√≥n de c√≥mo se distribuyen las categor√≠as en el conjunto de datos. Los diagramas de caja (boxplots) son otra herramienta excelente para visualizar datos num√©ricos. Estos diagramas muestran la mediana de los datos, los cuartiles y los valores at√≠picos. La caja central representa el rango intercuart√≠lico, es decir, los valores entre el cuartil 25 y el cuartil 75. Los puntos fuera de los extremos de la caja indican valores at√≠picos.

Los diagramas de caja son especialmente √∫tiles para comparar la distribuci√≥n de datos entre diferentes grupos. Por ejemplo, en un conjunto de datos de coches, podemos usar un diagrama de caja para comparar el precio de los coches seg√∫n el tipo de tracci√≥n. Al hacerlo, podemos observar que el precio de los coches con tracci√≥n trasera se distribuye de manera diferente al de los coches con tracci√≥n delantera o a las cuatro ruedas. Esto puede ser indicativo de una influencia de la tracci√≥n en el precio del coche.

A menudo, tambi√©n nos encontramos con **variables continuas** en los datos, como el precio y el tama√±o del motor de un coche. Para explorar la relaci√≥n entre estas dos variables, podemos usar un **diagrama de dispersi√≥n (scatter plot)**. Este tipo de gr√°fico nos permite visualizar la relaci√≥n entre dos variables. En este caso, el tama√±o del motor es la variable predictora (en el eje X) y el precio es la variable objetivo (en el eje Y). Al observar el diagrama de dispersi√≥n, podemos notar que, en general, a medida que aumenta el tama√±o del motor, tambi√©n lo hace el precio del coche, lo que sugiere una relaci√≥n lineal positiva entre ambas variables. Es importante recordar etiquetar los ejes y a√±adir un t√≠tulo a la gr√°fica para una correcta interpretaci√≥n de los datos.

# Correlaci√≥n


En este texto, hablaremos de la correlaci√≥n entre diferentes variables. La correlaci√≥n es una m√©trica estad√≠stica utilizada para medir en qu√© medida las variables son interdependientes. En otras palabras, cuando observamos dos variables a lo largo del tiempo, nos interesa saber c√≥mo un cambio en una de ellas afecta al cambio en la otra.

Por ejemplo, se sabe que fumar est√° relacionado con el c√°ncer de pulm√≥n, ya que quienes fuman tienen una mayor probabilidad de desarrollar esta enfermedad. De manera similar, existe una correlaci√≥n entre las variables paraguas y lluvia. Cuando hay m√°s precipitaciones, m√°s personas usan paraguas, mientras que si no llueve, es probable que las personas no lleven uno. Por lo tanto, podemos afirmar que los paraguas y la lluvia son interdependientes y est√°n correlacionados. Sin embargo, es importante destacar que la correlaci√≥n no implica causalidad. Aunque sabemos que existe una correlaci√≥n entre el uso del paraguas y la lluvia, no tenemos suficiente informaci√≥n para determinar si el uso del paraguas causa la lluvia o si la lluvia es la que provoca el uso del paraguas.

En el campo de la ciencia de datos, normalmente nos enfocamos m√°s en la correlaci√≥n que en la causalidad. Veamos un ejemplo de correlaci√≥n entre el tama√±o del motor de un veh√≠culo y su precio. Al visualizar estas dos variables mediante un diagrama de dispersi√≥n, podemos agregar una l√≠nea de regresi√≥n, que muestra la relaci√≥n entre ambas. El objetivo principal de este gr√°fico es observar si el tama√±o del motor influye en el precio del veh√≠culo. En este caso, la l√≠nea que atraviesa los puntos de datos es bastante pronunciada, lo que indica que existe una relaci√≥n lineal positiva entre el tama√±o del motor y el precio. Es decir, a medida que aumenta el tama√±o del motor, tambi√©n lo hace el precio, lo que demuestra una correlaci√≥n positiva entre ambas variables.

Otro ejemplo ser√≠a la relaci√≥n entre las millas de carretera por gal√≥n y el precio del autom√≥vil. Al observar este gr√°fico, podemos ver que cuando el valor de las millas por gal√≥n aumenta, el precio disminuye. Esto muestra que existe una relaci√≥n lineal negativa entre estas dos variables. Aunque la relaci√≥n es negativa, la pendiente de la l√≠nea es pronunciada, lo que sugiere que las millas por gal√≥n siguen siendo un buen indicador del precio. Por lo tanto, se dice que estas dos variables tienen una correlaci√≥n negativa.

Por √∫ltimo, tenemos un ejemplo de una correlaci√≥n d√©bil. En este caso, tanto las RPM de pico bajas como las altas est√°n asociadas con precios bajos y altos, respectivamente. Esto indica que no podemos utilizar las RPM para predecir el precio de manera confiable.

### Introducci√≥n

La prueba de chi-cuadrado es un m√©todo estad√≠stico utilizado para determinar si existe una asociaci√≥n significativa entre dos variables categ√≥ricas. Esta prueba se utiliza ampliamente en diversos campos, incluyendo las ciencias sociales, el marketing y la atenci√≥n sanitaria, para analizar datos de encuestas, resultados experimentales y estudios observacionales.

### Concepto

La prueba de chi-cuadrado es un m√©todo estad√≠stico no param√©trico utilizado para examinar la asociaci√≥n entre dos variables categ√≥ricas. Eval√∫a si las frecuencias de los resultados observados se desv√≠an significativamente de las frecuencias esperadas, asumiendo que las variables son independientes. La prueba se basa en la distribuci√≥n chi-cuadrado, que se aplica a datos de conteo y ayuda a determinar si alguna desviaci√≥n observada podr√≠a haber surgido por azar.
